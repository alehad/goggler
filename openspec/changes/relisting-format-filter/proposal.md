## Why

The Home Relistings view can now include both auction and buy-now candidates. The user needs a quick way to narrow relistings by listing format without affecting the rest of the Home feed.

## What Changes

- Add a small listing-format segmented control that is visible only when the Home `Relistings` filter is active.
- Let the user show both relisting formats, auction-only relistings, or buy-now-only relistings.
- Keep the format control separate from the main Home filter control and aligned to the right edge of the listed items area.

## Out Of Scope

- Persisting the selected listing-format filter across sessions.
- Applying the listing-format filter to Search, Watchlist, Won, Never won, or All views.
